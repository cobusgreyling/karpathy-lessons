# Contributing

This is a curated, interpretive field guide — not an official record of Andrej Karpathy's views.
That said, accuracy matters. Contributions are welcome:

- **Corrections** — if a lesson misrepresents a source, or a claim has dated badly, open an issue
  or a PR. Cite the original essay, talk, or tweet.
- **Better sources** — if you can replace a secondary link with Karpathy's primary one, please do.
- **New lessons** — keep the voice tight and declarative, one idea per bolded paragraph, and add a
  `## Sources` block linking the original material. Match the structure of the existing chapters.

Each chapter lives in `docs/`. The site landing page is `docs/index.md`; the repo front page is
`README.md`. If you add or rename a chapter, update both the contents list and the prev/next
navigation footers.

Keep it honest: when in doubt, link the source and let readers judge.
